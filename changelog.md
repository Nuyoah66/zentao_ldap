# 禅道开源版 LDAP/AD 插件更新日志

## 版本 1.1 - 2025-04-26
### 禅道版本
18.13
### 更新内容
1. 修复 LDAP 用户邮箱为空时，同步用户导致同步失败的问题

## 版本 1.0 - 2025-04-25
### 禅道版本
18.13
### 更新内容
1. 支持 18.13 版本
2. 支持从 LDAP 服务器上同步用户信息（账号、真实姓名、邮箱）
3. 支持使用 LDAP 用户登录禅道
4. 支持本地用户，通过在账户名称前加 “$” 符号来登录禅道
5. 禁止 LDAP 用户修改账号密码
6. 禅道删除的用户与 LDAP 用户存在同名账户时，同步 LDAP 用户时，会启用账户并更新信息
